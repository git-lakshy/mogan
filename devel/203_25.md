# [203_25] 支持 \bold 和 \italic

## 如何测试
创建一个 tex 文件，内容如下：
```LaTeX
\documentclass{article}
\begin{document}

这是一个包含 \bold{粗体文本} 和 \italic{斜体文本} 的示例句子。

\end{document}

```
删除缓存：
```bash
rm -rf ~/.local/share/moganlab/
rm -rf ~/.cache/MoganLab/
```
启动 Mogan，文件->导入->LaTeX，选择刚刚创建好的 LaTeX 文件，查看 “粗体文本” 和 “斜体文本” 是否加粗与倾斜。
